# Rust_Projects
Few Projects in Rust for learning purpose.

## Few Cargo Commands to get started :

Build commands : 

    cargo bench
    cargo build
    cargo check
    cargo clean
    cargo doc
    cargo fetch
    cargo fix
    cargo run
    cargo rustc
    cargo rustdoc
    cargo test
   
Manifest Commands :

    cargo generate-lockfile
    cargo locate-project
    cargo metadata
    cargo pkgid
    cargo tree
    cargo update
    cargo vendor
    cargo verify-project
